E-Sangrahan is a initiative for creating awarness amongst the people on how import E-Waste is and how to deal with it. This is a software cum Management system which helps people dispose their E-Waste to us, we would then recycle it and in exchange you get credits which could be redeemed for non electronic gadgets. It has an E-Waste Identifier and locater for making the ease of use for customers.
